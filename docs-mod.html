CMD ShortC Input /
[Optional]		 Result
EnableCMD ECMD COMMAND Enable commands in Curiosity
;DisableCMD ;Disable COMMAND Disable commands in Curiosity
Ban B @user Reason Ban a specific user. Reason is only needed when you want to log mod commands into a channel
Bump Advertise your server to other servers
Flush FL 1 / 100 Delete x amount of bot messages
Kick K @User Reason Kick a specific user. Reason is only needed when you want to log mod commands into a channel
Leaderboard LB Returns a leaderboard of the most active servers with Curiosity
Lockdown LD Lockdown a channel. This is a anti-raid feature. This at the moment only works on the @everyone tag.
Purge P Delete x amount of messages
Unban U UserID Delete x amount of messages
VoiceMute VM @user Server voice mute a specific user
VoiceUnmute VUM @user Server voice unmute a specific user
Warn W @user Reason Warn a specific user. Reason is only needed when you want to log mod commands into a channel
