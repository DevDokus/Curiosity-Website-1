Changelogs - May 1st 2017
-----------------------------------------------------------------
Help Menu Changed !
We have changed the help menu completely !!!
The old help DM became to big as we are adding more
The new help menu is completely interactive !!

How does it work ?
you type ;Help - ;H on chat and a menu will spawn.
Now the only thing you have to do is typing the command in chat
to let the menu trigger the next window. No Prefix !!
-----------------------------------------------------------------

Multiple Features added to a Database !
Features as ;Warn - ;Kick - ;Ban - ;Stab - ;Spank
were before counted as server only. From now on all the above
features will also be counted Globally.
This means that if you use one of the above featues the member
(example warn) will receive a warning for both server and global.
Global means for all servers Curiosity is on !!
